Toolbox

The Toolbox is a set of static methods to extract some data from a parsed NRRD using the header and/or the data as returned by nrrdjs.parse(...).

The NRRD format does not make any assumption about the naming of the axis (X, Y, Z, A, B, C, etc.) but for the sake of accessibility, the Toolbox assumes that if there is more than 1 components per voxel (ex: RGB), they are encoded on the fastest axis. Otherwise:

  • The axis called X is encoded on the fastest axis
  • The axis called Z is encoded on the slowest axis
  • The axis called Y is encoded in between
  • The time axis, if any, is even slower than Z

Note: the fast axis is the one where element along it are contiguous on the buffer (stride: 1)
